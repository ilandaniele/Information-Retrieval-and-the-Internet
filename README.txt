HOW TO RUN:
In the console just write "python gui.py"
Then you will see the options to choose the model and which collection to search on. It's only available courses collection 
for now and you can choose between Vector Space Model and Boolean Model.
You can type down the query and it will lead you to the results, and then you click one and it will show its content in another window.
You can also change the options for stemming, normalization and stopwords removal, which is inside the code, 
and is just changing some values from 0 to 1.
CAUTION: Some libraries should be installed before using it:
 * pythonds
 * nltk
 * BeautifulSoup
 * tkinter
